### 介绍

Toa-UI 是一个基于 `React` 开发的电商移动端组件库，特点是轻量、依赖少、开箱即用。

### 特性

- 🚀 提供 200 + 个高质量通用组件、20+互动营销组件
- 🚀 组件十分轻量
- 💪 由国人开发，完善的中英文文档和后勤保障
- 🛠️ 支持按需引入
- 🛠️ 支持主题定制
- 🌍 支持国际化

### 安装

### CDN

`toa-ui.js` 包含组件库的所有样式和逻辑，引入即可。

### Webpack / Vite

```shell
# 通过 npm 或 yarn 或 pnpm 安装
# npm
npm i @toa/ui -S
# yarn
yarn add @toa/ui
# pnpm
pnpm add @toa/ui
```
