### 介绍

toa-ui 是一个电商风格移动端组件库，全面拥抱 `Vue3` 生态。

### 特性
- 🚀 提供50+个高质量电商通用组件，并在欧莱雅电商应用内20+品牌所使用 
- 🚀 组件十分轻量
- 💪 由国人开发，完善的中英文文档和后勤保障
- 🛠️ 支持按需引入
- 🛠️ 支持主题定制
- 🌍 支持国际化
- 💡 支持 webstorm，vscode 组件属性高亮
- 💪 支持 SSR
- 💡 支持 Typescript 
- 💪 确保90%以上单元测试覆盖率，提供稳定性保证
